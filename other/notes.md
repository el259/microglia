### How to run (Mack)

In terminal:


pyenv versions
- this will print the version-number

python3 -m venv simulation-env

source simulation-env/bin/activate

pyenv version X 
- this is the version-number from before

pip install LIST-PACKAGES

WHEN FINISHED, TYPE THE FOLLOWING IN TERMIANL:

deactivate

### Notes and Comments for Linux and/or Python Virtual Environments
If you are using Linux it may be required to use a Python virtual environment, which allows you to install packages and libraries that may not already be installed if you are running the simulation locally.

python3 -m venv simulation-env

source simulation-env/bin/activate

pip install LIST-PACKAGES
